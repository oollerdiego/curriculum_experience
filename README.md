# 📄 Currículo Online - Diego Oller

Este é um projeto de currículo estruturado inteiramente em **HTML5** e **CSS3** puros, desenvolvido como parte dos meus estudos em **Análise e Desenvolvimento de Sistemas**.

## 🚀 Objetivo
O foco deste projeto foi aplicar conceitos de layout responsivo, manipulação de cores e tipografia, além de tentar organizar minha trajetória acadêmica e profissional de forma moderna e acessível diretamente pelo navegador.

## 🛠️ Tecnologias Utilizadas
*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização utilizando Flexbox, unidades relativas (vw, vh, rem) e design de duas colunas.
*   **GitHub Pages:** Hospedagem e deploy automatizado.

## 📌 Destaques do Projeto
*   **Layout Adaptável:** Uso de `min-height` e `max-width` para tentar permitir uma boa leitura em diferentes tamanhos de tela.
*   **Estética Profissional:** Contraste planejado entre as seções de informações pessoais e conteúdo principal.
*   **Seção de Projetos:** Listagem de experiências anteriores, incluindo aplicações com Python e projetos premiados em feiras científicas (Cientec UFABC).
